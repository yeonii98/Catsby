# Catsby

### 프로젝트 개요
* QR코드를 활용한 길고양이 사료 급여 스케줄 관리 앱 입니다.

### 담당한 기능 (Android & BE)
* 동네 커뮤니티 구현
    - 커뮤니티 글 쓰기, 수정, 삭제
    - 좋아요 및 댓글
    - 동네별로 글 가져오기
* 내 글 목록
    - 내가 쓴 글, 내가 쓴 댓글
* 로그인
    - 페이스북 로그인
